# 🌸 Flora Mind — AI Bouquet Generator

> Веб-сервис для интеллектуальной генерации цветочных букетов на основе искусственного интеллекта

---

## 📌 О проекте

**Flora Mind** — это веб-приложение, которое помогает пользователю подобрать персонализированный букет без знаний во флористике.

Пользователь указывает:
- 🎉 **Повод** — день рождения, свидание, свадьба, извинение и др.
- 👤 **Получателя** — партнёр, друг, коллега
- 🎨 **Настроение и предпочтения**

На основе этих данных ИИ:
- Генерирует **визуальную концепцию букета** (подбор цветов, количество, стиль композиции)
- Создаёт **романтичное текстовое описание** для карточки товара или открытки

## 🚀 Запуск проекта

### Клиентская часть
```bash
cd frontend
npm install
npm start
```

### Серверная часть
```bash
cd backend
npm install
node server.js
```
